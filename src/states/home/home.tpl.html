Welcome To Codexen

Features

  • Twitter + Github

    You can find codes by tag, follow other users, and make a team. Also, you can fork every codes with *A single Click*.

  • Terminal Friendly

    CodeXen offer you CLI Tool to fetch your codes. You can generate or run directly it with your *Fully Customized Command*.

  • Mobile First

    This App is available to use on every device. In CodeXen, you can browse codes and set up your commands with your *Smart phone*. :)

How to Use

Tutorial

  • Install

    You can install via npm(Node package manager). 

    npm install -g codexen

  • Set Up

    First of all, you should fetch your Depot(command set) from server. 

    cxcx init {depot_name}
    Default depot name is your username : laravelfanatic The depot data will be fetched to codexen.json

  • Use

    You are ready to go. You can use all Commands you registered.

    cx {command}
    You can register commands here

    If you use generator, you can override filename with the second parameter

    cx {command} {override_filename}

    Given :

    There is a command, 'idx', to generate 'index.js'

    Without the second parameter
    cx idx

    index.js generated.

    With the second parameter
    cx idx yolo

    yolo.js generated.

  • Update commands

    If some changes occur, you can re-fetch the changed commands easily. 

    cxcx update
